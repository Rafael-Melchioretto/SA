# Estudo da Identidade Visual – Sistema Ferroviário Pampa-Serra

A identidade visual do sistema ferroviário Pampa-Serra foi criada para transmitir modernidade, tecnologia, segurança e organização. O design do sistema busca facilitar o monitoramento ferroviário em tempo real de forma simples e intuitiva.

**Cores**

As principais cores utilizadas são:

- Azul escuro: representa tecnologia, confiança e estabilidade;
- Branco: transmite limpeza visual e facilita a leitura;
- Cinza claro: utilizado no fundo para melhorar o conforto visual;
- Verde, amarelo e vermelho: usados nos alertas de status e monitoramento.

**Tipografia**

Foi utilizada uma fonte sem serifa, moderna e de fácil leitura, com destaque para títulos e menus em negrito, facilitando a navegação do usuário.

**Estilo Visual**

O sistema possui um estilo:
- Moderno;
- Minimalista;
- Tecnológico;
- Funcional.

Os elementos utilizam bordas arredondadas, ícones simples e organização limpa para melhorar a experiência do usuário.

**Referências Visuais**

A interface foi inspirada em:
- Sistemas de monitoramento;
- Aplicativos de mobilidade urbana;
- Painéis administrativos modernos;
- Sistemas ferroviários inteligentes.

Os ícones de trem, sensores e relatórios reforçam a proposta tecnológica do projeto.

**Proposta de Layout**

O layout foi desenvolvido para ser intuitivo e organizado.

**Tela de Login**
Possui logo em destaque, campos simples de acesso e botão de entrada centralizado.

**Menu Principal**
Conta com botões grandes e objetivos para acesso rápido às funções:
- Rotas;
- Sensores;
- Relatórios;
- Cadastro.

**Tela de Monitoramento**
Exibe mapa da rota ferroviária, velocidade, consumo, alertas e localização do trem em tempo real.

**Objetivo**

A identidade visual do Pampa-Serra busca criar um sistema moderno, funcional e de fácil utilização, facilitando o gerenciamento ferroviário inteligente.
