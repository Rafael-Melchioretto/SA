# Aplicação do CRUD no Sistema Pampa-Serra

No sistema ferroviário Pampa-Serra, o CRUD é utilizado para gerenciar sensores, usuários, rotas e informações da malha ferroviária em tempo real.

**Create (Criar)**

A operação Create é responsável pelo cadastro de novos dados no sistema.

**Exemplos:**
Cadastro de novos usuários na tela de cadastro;
Adição de sensores ferroviários;
Registro de novos dispositivos de monitoramento.

Ao clicar em botões como “Adicionar” ou “Cadastrar”, o sistema cria uma nova informação no banco de dados.

**Read (Ler)**

A operação Read permite visualizar informações já registradas.

**Exemplos:**
Visualização das rotas ferroviárias;
Leitura da velocidade, localização e consumo do trem em tempo real;
Exibição de relatórios e gráficos;
Listagem dos sensores instalados nos trilhos.

O sistema utiliza essa operação constantemente para monitoramento e acompanhamento das viagens.

**Update (Atualizar)**

O Update é utilizado para modificar informações existentes.

**Exemplos:**
Atualização do status dos sensores;
Alteração de dados de monitoramento;
Mudança de alertas de operação;
Atualização automática das informações de tráfego ferroviário.

Quando o sistema altera um status de “Esperado” para “Atenção”, por exemplo, ocorre uma atualização dos dados.

**Delete (Apagar)**

A operação Delete remove informações que não são mais necessárias.

**Exemplos:**
Exclusão de sensores obsoletos;
Remoção de registros incorretos;
Exclusão de dispositivos da malha ferroviária.

Nas telas do sistema, essa função é representada pelos ícones vermelhos de subtração (-) presentes nos cards dos sensores.

**Conclusão**

O CRUD é essencial para o funcionamento do sistema Pampa-Serra, permitindo cadastrar, visualizar, atualizar e remover informações da ferrovia de forma organizada e eficiente.
